## 实现功能
+ 邮箱注册
+ 登录
+ 发布商品
+ 购买商品
+ 浏览商品列表
+ 聊天沟通
+ 消息列表
## 首页
首页实现了一些简单的功能,随即展示所有发布的商品,搜索功能暂未实现。顶部是一个搜索栏和二维码扫描按钮，中间是一个Banner轮播图，用于展示活动图片，下面是一个商品子类选择，可以选择不同的子类，最下面是每个子类下随机商品展示。
