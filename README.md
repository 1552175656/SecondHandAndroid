# 仿闲鱼的二手交易平台-安卓端
** [用Django写的服务端](https://github.com/zstu-lly/SecondHandDjango/blob/master/README.md)
## 实现功能
+ 邮箱注册
+ 登录
+ 发布商品
+ 购买商品
+ 浏览商品列表
+ 聊天沟通
+ 消息列表
